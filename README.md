# Opencraft
This is the repository to an open-source Minecraft clone. Credit to Mojang Studios for the fonts, graphics, and audio.

# Forking
You are able to fork this, if you like. However, please don't remove any credits for Mojang Studios, and you may include a link to your fork's repository, but please don't remove the link to the original repository.

# Assets
Textures from Minecraft Bedrock Edition. Leaf texture converted from .tga to .png via [CloudConvert](https://cloudconvert.com/tga-to-png). Background image is a screenshot of the Opencraft gameplay. Audio from [Minecraft Wiki](https://minecraft.wiki).
